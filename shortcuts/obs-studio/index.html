OBS Studio Keyboard Shortcuts Guide

OBS Studio hotkeys are one of the fastest ways to control your stream without clicking UI buttons. Streamers commonly search for OBS hotkeys for scene switching, OBS start/stop recording shortcut, and OBS mute mic shortcut. Use this page as a quick reference and then set your own hotkeys in Settings → Hotkeys.

Best OBS hotkeys to set first

  • Start/Stop Recording
  • Start/Stop Streaming
  • Mute Mic and Mute Desktop Audio
  • Scene switching (1–9)
  • Studio Mode Transition

Pro tip: avoid conflicts

Choose combinations that don’t conflict with your game, Discord, or OS shortcuts. Test hotkeys in a private recording session before going live.

OBS Studio Shortcuts — Frequently Asked Questions

How do I set up hotkeys in OBS Studio?

Go to Settings → Hotkeys and assign keyboard shortcuts to any OBS action—start streaming, switch scenes, mute mic, and more.

What are the default OBS Studio keyboard shortcuts?

OBS has no universal defaults—hotkeys are user-defined. Common setups bind F1-F12 to scene switches and Ctrl+Shift+S to start/stop streaming.

Can I use OBS hotkeys while gaming?

Yes, if you enable 'Hotkeys focus behavior' appropriately. Global hotkeys work even when OBS is not the active window.

Are OBS shortcuts the same on Mac and Windows?

OBS hotkeys are fully customizable and platform-independent once configured in Settings → Hotkeys.
