Mastering Figma Keyboard Shortcuts in 2025

Figma keyboard shortcuts are essential productivity tools that transform how designers, product managers, and creative teams work on UI/UX design, prototyping, and collaborative projects. With over 200,000 monthly searches for "Figma shortcuts," mastering these hotkeys is crucial for anyone serious about modern design workflows. This comprehensive guide covers 150+ essential Figma keyboard shortcuts for Windows, Mac, and Web, helping you design up to 50% faster in Figma.

Why Learn Figma Keyboard Shortcuts?

Learning Figma keyboard shortcuts dramatically improves design efficiency, reduces repetitive tasks, and enables faster prototyping workflows. UI/UX designers, product designers, graphic designers, and design teams rely on shortcuts for tasks like frame creation, component manipulation, boolean operations, and real-time collaboration. Studies show that designers who master Figma shortcuts complete projects 40-60% faster than those relying solely on mouse navigation. Just like Photoshop keyboard shortcuts help designers edit faster, Figma shortcuts are essential for modern interface design workflows.

Essential Figma Tool Shortcuts

The most critical Figma shortcuts are tool selection keys. Press V for Move tool, F for Frame tool, R for Rectangle, O for Ellipse, and T for Text tool. These single-key shortcuts eliminate menu navigation and keep you in your creative flow. Press P for Pen tool to create custom vectors, L for Line tool, and K for Scale tool. Master these tool shortcuts first, as they form the foundation of efficient Figma usage and are used hundreds of times per design session.

Frame and Object Manipulation

Figma's frame-based design system makes frame shortcuts incredibly powerful. Press Ctrl+Alt+G (Cmd+Option+G on Mac) to create frames from selections. Duplicate objects instantly with Ctrl+D, and group elements with Ctrl+G. Use Ctrl+Shift+H to hide/show layers, Ctrl+Shift+L to lock layers, and Ctrl+] or Ctrl+[ to adjust layer order. These shortcuts enable rapid layout construction and design iteration without breaking your creative momentum.

Component and Auto Layout Magic

Figma's component system revolutionizes design systems, and shortcuts make it lightning-fast. Create components with Ctrl+Alt+K, detach instances with Ctrl+Alt+B, and create component sets for variants. Add auto layout with Shift+A to create responsive designs instantly. Adjust padding and spacing with Alt+Arrow keys while in auto layout. These advanced shortcuts separate Figma beginners from power users, enabling scalable design system creation.

Boolean Operations and Vector Editing

Vector editing in Figma becomes powerful with boolean operation shortcuts. Press Ctrl+Alt+U for union, Ctrl+Alt+S for subtract, Ctrl+Alt+I for intersect, and Ctrl+Alt+X for exclude. Enter vector edit mode by pressing Enter on any shape, then use A to add points and Delete to remove them. These shortcuts enable complex icon design, logo creation, and custom UI element crafting directly in Figma.

Prototyping and Interaction Shortcuts

Figma's prototyping features shine with keyboard shortcuts. Press Shift+E to enter prototype mode, then drag connections between frames for interactions. Preview prototypes instantly with Ctrl+Alt+Enter (Cmd+Option+Enter on Mac). Navigate prototype flows with arrow keys during presentation mode. Copy prototype links with Ctrl+Shift+L for stakeholder sharing. These shortcuts streamline the entire prototyping workflow from design to user testing.

Collaboration and Commenting

Figma excels at real-time collaboration, and shortcuts enhance teamwork. Press C to add comments anywhere on your design, @ to mention teammates for instant notifications, and Ctrl+/ to show/hide UI for focused design reviews. View version history with the keyboard and restore previous iterations. These collaboration shortcuts keep design teams aligned and productive. Similar to how Notion shortcuts facilitate team knowledge sharing, Figma shortcuts enable seamless design collaboration.

Search and Navigation Power Moves

Navigate massive Figma files effortlessly with search shortcuts. Press Ctrl+/ (Cmd+/ on Mac) to open quick actions and search everything in Figma. Use Ctrl+P to search layers, pages, and frames instantly. Press 1 to zoom to 100%, 2 to zoom to fit, and 0 to zoom to selection. Navigate between pages with Page Up/Down, and between frames with N key. These navigation shortcuts prevent getting lost in complex design files.

Plugin and Resource Shortcuts

Figma's plugin ecosystem extends functionality, and shortcuts make plugins accessible. Press Ctrl+Alt+P (Cmd+Option+P on Mac) to run plugins instantly. Access recent plugins from the quick menu. Many popular plugins like Unsplash, Icons8, and Content Reel have their own keyboard shortcuts. Master plugin shortcuts to supercharge your workflow with specialized design tools without leaving Figma.

Design Systems and Styles

Maintaining consistent design systems becomes effortless with Figma style shortcuts. Copy styles with Ctrl+Alt+C and paste with Ctrl+Alt+V. Create color styles, text styles, and effect styles with shortcuts for instant reusability. Update shared library components simultaneously across projects. These design system shortcuts ensure brand consistency and reduce repetitive styling tasks across large projects.

Platform-Specific Shortcuts

Figma shortcuts work across Windows, Mac, and web browsers with slight variations. Windows and Linux use Ctrl as the primary modifier, while Mac uses Cmd (‚åò) and Option (‚å•) instead of Alt. Most shortcuts translate directly: Ctrl+D vs Cmd+D for duplicate, Ctrl+G vs Cmd+G for group, etc. The web app supports all desktop shortcuts through browser compatibility. Mobile apps offer touch gestures for common actions but lack full keyboard support.

Tips for Learning Figma Shortcuts

Start by memorizing the "Essential 7" Figma shortcuts: V (Move), F (Frame), R (Rectangle), T (Text), Ctrl+D (Duplicate), Ctrl+G (Group), and Shift+A (Auto Layout). Practice these during every design session until they become muscle memory. Gradually add more shortcuts weekly, focusing on your most repeated actions. Enable Figma's shortcut hints in settings to learn while you work. Create a personal cheat sheet of your favorite shortcuts. Most importantly, force yourself to use keyboard shortcuts instead of toolbar clicking.

Integration with Design Workflow

Figma shortcuts become even more powerful when combined with your complete design stack. Use browser shortcuts to research inspiration quickly. Combine Figma with VS Code shortcuts for design-to-development handoff workflows. Integrate with project management tools while maintaining keyboard-driven efficiency. Many professional designers master shortcuts across their entire tool ecosystem, creating seamless workflows from concept to shipped product.
