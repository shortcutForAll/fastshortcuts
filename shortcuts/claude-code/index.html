Claude Code Keyboard Shortcuts

Speed up Claude Code with practical shortcuts for prompt editing, command history, session control, and slash commands. Use this with our Cursor shortcuts and VS Code shortcuts pages for complete AI coding workflows, and pair it with Perplexity shortcuts for faster research-to-code loops.

What’s Covered in This Claude Code Cheat Sheet

This guide includes terminal control keys, text-editing keys, history search, session actions, and frequently used slash commands. Some bindings can vary by terminal emulator, shell profile, or app settings, so verify the keys in your own setup. For source discovery and citation workflows before coding, keep our Perplexity keyboard shortcuts reference open in a second tab.
