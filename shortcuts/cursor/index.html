Cursor IDE Keyboard Shortcuts

Speed up the Cursor AI code editor with shortcuts for chat, Agent, inline edit, context, and the integrated terminal. Cursor extends VS Code—defaults below follow Cursor’s keyboard docs; on Windows/Linux, Cmd maps to Ctrl and Opt to Alt in most chords. For editor baseline keys, use our VS Code guide; for JVM workflows, see IntelliJ shortcuts.
