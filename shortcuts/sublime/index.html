Sublime Text Shortcuts

Every essential Sublime Text shortcut—multi-cursor editing, Goto Anything, command palette, and code folding. Reference: Sublime Text official key bindings.

Why Sublime Text Shortcuts Matter (2026)

Sublime's killer feature is multi-cursor editing: Ctrl/Cmd+D selects the next match, Ctrl/Cmd+Shift+L selects all matches. Cmd/Ctrl+P opens Goto Anything; Cmd/Ctrl+Shift+P opens the command palette. Still a favorite for speed-focused developers alongside VS Code and Zed.

Sublime Text Shortcuts — Frequently Asked Questions

What is the most important Sublime Text shortcut?

Cmd/Ctrl+P (Goto Anything) is the single most powerful shortcut—it jumps to files, symbols, and line numbers instantly.

How do I use multiple cursors in Sublime Text?

Press Cmd/Ctrl+D to select the next occurrence of the current word. Cmd/Ctrl+Shift+L selects all occurrences at once. Hold Alt and click to add cursors manually.

How do I open the command palette in Sublime Text?

Press Cmd+Shift+P on Mac or Ctrl+Shift+P on Windows/Linux to open the command palette and run any Sublime command by name.

Can I customize Sublime Text key bindings?

Yes. Edit Preferences → Key Bindings to add custom shortcuts in JSON. Sublime uses a flexible keymap system similar to VS Code.
