Claude Keyboard Shortcuts

Every Claude keyboard shortcut for 2026 in one cheat sheet - native claude.ai web shortcuts, the Claude Desktop apps for Mac and Windows, Projects, Artifacts, voice mode, search palette (Ctrl + K / Cmd + K), slash commands, and 100+ power-user keys. Looking for the terminal CLI? See our Claude Code shortcuts. Pair this guide with our ChatGPT shortcuts, Perplexity shortcuts for research, and editor cheat sheets for Cursor, Windsurf and VS Code.

What this Claude shortcut guide covers

Anthropic does not publish a single dedicated keyboard shortcut page for claude.ai. The reality is the shortcut surface is split across three different products - the web app at claude.ai, the Claude Desktop apps for macOS and Windows, and the Claude mobile apps with iOS Shortcuts and App Intents. This guide pulls all of them into one place, plus the browser-level chords and OS shortcuts that genuinely speed Claude up day-to-day.

  • Native claude.ai web shortcuts (new chat, sidebar, copy response, search palette, slash commands)
  • Claude Desktop on Mac and Windows - global hotkey, window controls, zoom, settings
  • Artifacts panel, Projects, memory and chat search shortcuts
  • Voice mode, file attach and dictation hotkeys
  • Browser-level shortcuts that supercharge Claude.ai inside Chrome, Edge, Safari and Firefox
  • Markdown formatting helpers so your prompts render exactly the way Claude expects
  • Mobile (iOS / Android) - what works with an external keyboard and where iOS Shortcuts fit in

A small but important detail: most claude.ai shortcuts only fire when the page has focus and your cursor is outside the message input. If a chord does nothing, click on the conversation area first, or focus the input with Ctrl + Shift + ; (Windows / Linux) or Cmd + Shift + ; (Mac) and try again. On Claude Desktop, the same shortcuts work plus the native window controls below.
