Linear Keyboard Shortcuts

Master Linear—the keyboard-first issue tracker built for high-performance engineering teams. Every shortcut below is sourced from Linear's official documentation. Linear is designed to be used almost entirely without a mouse: single-key shortcuts for issue creation and management, G chords for navigation, and Cmd+K / Ctrl+K for the full command menu. If you're migrating from Jira, see our Notion and Slack guides for related team tools.

Why Linear Shortcuts Matter (June 2026)

Linear processes keyboard shortcuts in under 50ms — faster than any UI click. The tool is explicitly built around the idea that every common action should be reachable in two keystrokes or fewer. The single most important shortcut to memorize is Cmd+K / Ctrl+K (command menu): if you forget any other shortcut, open the command menu and type what you want to do. The second most important is C (create issue): it works from anywhere in the app. Start with those two, then layer in the G + letter navigation chords as you build muscle memory. Teams migrating from Jira or Asana consistently report that Linear's keyboard coverage is the biggest workflow upgrade — every status change, assignment, and priority update is one keypress away from any issue list.

Linear Shortcuts — Frequently Asked Questions

What is the most important shortcut in Linear?

Cmd+K on Mac or Ctrl+K on Windows opens the command menu, which gives you access to every action in Linear by name. If you forget any other shortcut, open the command menu and type what you want to do — change status, assign, move team, everything.

How do I create a new issue in Linear?

Press C from anywhere in Linear to open the new issue dialog. Press Alt/⌥ + C to create from a template. Press V for the full-screen issue editor. Save with Ctrl/⌘+Enter.

How do I navigate between views in Linear using the keyboard?

Use the G + letter chord to jump to any view instantly: G I = Inbox, G M = My Issues, G P = Projects, G T = Teams, G R = Roadmap, G C = Cycles. Press / to open global search.

How do I change issue priority or status with a shortcut?

With an issue focused, press S to open the status picker and P to open the priority picker. For quick priority: Shift+1 = Urgent, Shift+2 = High, Shift+3 = Medium, Shift+4 = Low.

Are Linear shortcuts the same on Mac and Windows?

All single-key shortcuts (C, S, P, ?, etc.) and G/O navigation chords are identical on both platforms. For modifier shortcuts, Mac uses Cmd (⌘) where Windows uses Ctrl, and Mac uses Option (⌥) where Windows uses Alt. Use the platform tabs above to switch views.
