Discord Keyboard Shortcuts

Navigate servers, DMs, and voice faster with Discord’s official shortcuts for desktop and browser (Windows, Linux & macOS).

General Navigation

Open User Settings
Ctrl,
Open Keyboard Shortcuts Menu
Ctrl/
Create or Join a Server
CtrlShiftN
Navigate Back
Alt←

Search & Pickers

Search in Current Channel
CtrlF
Search Across All Channels
CtrlShiftF
Open Emoji Picker
CtrlE
Open GIF Picker
CtrlG

Messages & Channels

Browse Channel Message History (older / newer)
Ctrl↑/Ctrl↓
Edit Your Last Message
Shift↑
Cancel Message or Mark Channel Read
Esc
Move Between Channels
Alt↑/Alt↓
Quick Switcher (Channels / DMs)
CtrlK

Voice & Servers

Mute / Unmute Microphone
CtrlShiftM
Deafen / Undeafen Audio
CtrlShiftD
Switch Between Servers
CtrlAlt←/CtrlAlt→

Text Formatting

Bold Text
CtrlB
Italic Text
CtrlI
Underline Text
CtrlU
